# tsSleepBlock001
====

Overview
CPU使用中はマシンを休止しない

## Description
コードサンプル用です。
CPU使用中はwindowsマシンを休止しない

## Usage
・ツールを起動
・監視時間を指定（分）
・CPUの閾値を指定（％）
・「START」ボタン押し下げ
